The Slogan Tutorial

Slogan allows you to program at a very high level using expressions and functions. Slogan obeys the venerable semantics of Scheme. This means that Slogan stands on solid ground and will be a pleasant and versatile tool - for both the experienced and the novice.

This tutorial will get you started programming in Slogan, as quickly as possible. All sample code can be directly entered at the REPL and evaluated. In the text of the tutorial, an expression or a program statement to be typed at the REPL is preceded by the prompt `>' and the result of evaluation will be prefixed by `=>'.

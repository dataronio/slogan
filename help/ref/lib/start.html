The Slogan Library Reference

The purpose of this document is to describe the standard functions and libraries that come packaged with Slogan. These functions can be categorized as follows:

  1. Equivalence predicates
  2. Numbers
  3. Pairs and Lists
  4. Strings and Characters
  5. Arrays, BitVectors and Tables
  6. Date and Time
  7. Streams
  8. Tasks
  9. Interpretation, evaluation and compilation
  10. Foreign Function Interface

Streams

stream (type, settings)
output_stream (type, settings)
input_stream (type, settings)

Returns a new stream object. type is a symbol that specifies the type of stream to create. The stream types supported in this implementation of Slogan are: !file, !array, !byte_array, !string, !process, !tcp_client, !tcp_server and !directory. settings can be either a string or a list of key-value settings that is dependent on the stream type. A stream created by stream can do both output and input.

with_stream (type, settings, fn)
with_input_stream (type, settings, fn)
with_output_stream (type, settings, fn)

Creates a stream and calls fn with the new stream object as argument. The stream is automatically closed before the function returns.

The Slogan Programming Language

Introduction

Slogan is a general purpose, dynamically typed, expression-oriented programming language. Slogan makes it easy to write multi-threaded, distributed applications. This implementation, which acts as the specification of the language, comes with an interpreter and an optimizing compiler that generates platform specific binaries.

Installing Slogan

You can fetch the latest source code of Slogan from github. Slogan should build on all modern Unixes - GNU/Linux, Mac and the various BSDs. Once you have cloned the git repository, issue these commands to compile Slogan:

            $ cd /path-to-slogan-directory
            $ ./configure-platform
            $ make

After the sources are compiled, type the following command to launch the Slogan interpreter:

$ ./src/slogan

Now you should see a prompt like this:

            slogan>

This is the Slogan REPL (Read-Eval-Print-Loop) where you can enter expressions. Slogan will print back the results of evaluating the expressions. Here are a few simple expressions for you to try:

            slogan> "hello, world";
            "hello, world"
            slogan> 1 + 2;
            3

To quit the REPL, type exit(); and press enter.

To compile Slogan programs from any directory, you have to create the environment variable `SLOGAN_ROOT' and point that to the directory where you have checked out the Slogan repository:

            $ export SLOGAN_ROOT=/path-to-slogan-directory

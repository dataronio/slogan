The Slogan Library Reference

The purpose of this document is to describe the standard functions and libraries that come packaged with Slogan. These functions can be categorized as follows:

  1. Equivalence predicates
  2. Functions
  3. Numbers
  4. Pairs and Lists
  5. Symbols, Characters and Strings
  6. Arrays, Byte arrays, Bit arrays and Tables
  7. Date and Time
  8. Streams
  9. Tasks
  10. Interpretation and compilation
  11. Control
  12. Foreign Function Interface

Note:As Slogan is derived from Scheme, large portions of the library specification is based on R5RS. Documentation of functions accessed from the Gambit platform is based on the Gambit manual.

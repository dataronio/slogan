Interpretation and compilation

compile (script_name, @key assemble, exe, output, ld_options, cc_options, exception_handler)

Compiles a file that contains Slogan code into machine code. If exe is true the object files are linked to produce a standalone executable. output is the name of the executable produced by the compiler. It defaults to script_name with no extension. ld_options is a string that contains the command-line options for the linker and cc_options is a string that contains the command-line options for the compiler. The compiler defaults to gcc. exception_handler is a function that takes a single exception object as argument. It defaults to display_exception.

eval (compiled_expr)
